# Web-Design
منهاج شامل ومدروس لدورة HTML5، مصمم ليأخذك من الصفر وحتى احتراف بناء هيكل صفحات الويب، مع مشروع تطبيقي في النهاية لتثبيت المعلومات.
الوحدة الأولى 
أساسيات الويب والـ HTML
•  كيف يعمل الويب، وما هو الفرق بين HTML و CSS و JavaScript.
• تجهيز بيئة العمل: تثبيت محرر الأكواد (مثل VS Code) وإضافة أدوات مساعدة (Extensions).
• الهيكل الأساسي: فهم الوسوم <!DOCTYPE html>، <html>، <head>، <body>.
• النصوص والعناوين: استخدام وسوم العناوين (<h1> إلى <h6>)، والفقرات (<p>).
• تنسيق النصوص: التعرف على وسوم التنسيق مثل <strong>، <em>، <br>، <hr>.

الوحدة الثانية
الروابط والوسائط المتعددة
• الروابط التشعبية: استخدام الوسم <a> للتنقل بين الصفحات والمواقع، وفهم خاصية href و target.
• الصور: إضافة الصور باستخدام <img>، وأهمية خواص src و alt (لتحسين محركات البحث).
• الصوت والفيديو: تضمين ملفات الميديا باستخدام وسوم HTML5 الحديثة <audio> و <video> مع خصائص التحكم controls.
• تضمين المحتوى الخارجي: استخدام <iframe> لعرض خرائط جوجل أو مقاطع يوتيوب داخل موقعك.

الوحدة الثالثة

القوائم والجداول
• القوائم: إنشاء القوائم المرتبة <ol> وغير المرتبة <ul> وعناصرها <li>.
• هيكل الجدول: بناء الجداول باستخدام <table>.
• صفوف وأعمدة الجدول: استخدام <tr> للصفوف، <td> للخلايا العادية، و <th> لعناوين الجدول.
• تقسيم الجدول: تنظيم الجداول المعقدة باستخدام <thead>، <tbody>، و <tfoot>.

الوحدة الرابعة 

النماذج (Forms) وإدخال البيانات
• هيكل النموذج: استخدام الوسم <form> للبدء.
• أنواع المدخلات (HTML5): التعرف على أنواع <input> المتعددة مثل: text، email، password، number، date، color.
• عناصر إضافية: استخدام <textarea> للنصوص الطويلة، و <select> للقوائم المنسدلة، و <button>.
• التحقق من صحة البيانات: استخدام الخصائص المدمجة في HTML5 مثل required، placeholder، min، max.

الوحدة الخامسة 

الدلالية (Semantic HTML5)
• أهمية الوسوم الدلالية: لماذا يجب التخلي عن كثرة استخدام <div> ولماذا هي مهمة لـ SEO (محركات البحث) و Accessibility (إمكانية الوصول).
• وسوم تخطيط الصفحة: استخدام <header>، <nav>، <main>، <section>، <article>، <aside>، و <footer>.
